# User Fask
Register and Collect User Data, Logging, Changes and Updates. Build with Python and Flask Microframework.

Run ```export LANG=en_US.UTF-8 LANGUAGE=en_US.en LC_ALL=en_US.UTF-8```

Run ```python3 -m flask run```
